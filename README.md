# 📌 Project Overview

This project analyzes Blinkit's sales performance using Python for data cleaning and Power BI for interactive dashboard creation.

The objective is to transform raw sales data into meaningful business insights that help understand customer purchasing behavior, outlet performance, product sales, and overall business growth.

```
      ▼
Business Insights
```

---

# 📂 Dataset Source

- Dataset downloaded from **Kaggle**
- Dataset contains sales, outlet, item, rating, and location information.

---

# 🧹 Data Cleaning Process

The dataset was cleaned using **Python** in **Google Colab**.

Cleaning steps included:

- Removed duplicate records
- Handled missing values
- Corrected data types
- Standardized column names
- Removed unnecessary columns
- Checked for inconsistent values
- Prepared data for visualization
- Exported the cleaned dataset as CSV

# 📊 Key Insights

- Supermarkets generate the highest sales.
- Tier 3 outlets contribute the largest share of revenue.
- Medium-sized outlets perform better than small outlets.
- Fruits & Vegetables and Snack Foods are top-selling categories.
- Average customer rating is approximately **3.92**.
- Total sales exceed **1.2 Million**.
